# Endpoint

Endpoint is a first-person API puzzle prototype built with Next.js, React Three Fiber, and Three.js.

## Run

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Controls

- `WASD` move
- `Mouse` look
- `Shift` sprint
- `Ctrl` or `C` crouch / slide
- `Space` jump
- `E` interact
- `Alt` toggle cursor mode
- `Esc` release pointer lock

## Prototype Goal

Walk to the API door keypad, press `E`, edit the request payload in the code editor, and run it to unlock the door.
